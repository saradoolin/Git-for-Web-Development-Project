1. What is Semantic HTML? 
    Semantic HTML is when we give elements meaning. For example, we use <p> for paragraphs and <a> for anchors.

2. What is HTML used for? 

    HTML is used to create the structure of a webpage, such as the sections, paragraphs, headings, etc. 

3. What is an attribute and where do we put it? 

    An attribute provides more information about an html element. We put it within the start tag.

4. What is the h1 tag used for? How many times should I use it on a page?

    H1 is for the main heading and should only use it once per page. 

5. Name two tags that have required attributes.
    
    The <a> tag has to have an attribute href="#" so that it will link to a destination. the <img> tag requires an attribute of src="#" to link to the image. 

6. What do we put in the head of our HTML document? 
    
    We put all the information about the website that we don't want displayed in the body, such as the language it is in, google fonts etc.

7. What is an id? 
    
    An id is the most specific selector that will override element and class selectors.

8. What elements can I add an id to? 
    
    You can add an id to any element.

9. How many times can I use the same id on a page? 

    You can only use the same id once. It is unique. 

10. What is a class? 

    A class is a selector that applies the same changes to any element that is given that class. 

11. What elements can I add a class to? 

    You can add class to any element.

12. How many times can I use the same class on a page? 

    You can use the same class a limitless amount of times. 

13. How do I get my link to open in a new tab?

    You have to apply the <a> tag then href the location of the new tab. For example, if you are on index.html and you want to go to contact.html, you would write it <a href="contact.html"></a>.

14. What is the alt attribute used for? 

    The alt attribute gives information about a picture for the sight impaired.

15. How do I reference an id?

    You reference an ID with the #.

16. What is the difference between a section and a div

    A section blocks out big areas of html, such as chapters, headers, etc, whereas divs are generic areas mainly used to help with CSS.

17. What is CSS used for? 

    CSS is used to style HTML. 

18. How to we select an element? Example - every h2 on the page
    
    You would just put the element name - in this case h2 {}.

19. What is the difference between a class and an id? - Give me an example of when I might use each one.

    Class can be used a limitless amount of time whereas id can only be used once. You apply class when you have a certain type of style you want for that element, like if I wanted all my <p> to have a font size of 1.6rem. If there is one <p> only I want to be larger than the rest, I could ID that paragraph so that I could make that one 2.0rem instead.

20. How do we select classes in CSS?

    We select a class with a period before the element. For example, first-element in the html would be selected in the CSS as .first-element. 

21. How do we select a p element with a single class of “human””?

    .human

22. What is a parent child selector? When would this be useful? 

    A parent child selector is when you can select a child within a parent by putting the child name after the parent name when you call it in the CSS file. This makes calling the elements a lot less hassle, and gets even better once you are able to nest with LESS.

23. How do you select all links within a div with the class of sidebar?

    .sidebar a {}

24. What is a pseudo selector?

    Pseudo selectors are a selector that selects elements that are in a specific state, such as hover. 

25. What do we use the change the spacing between lines?

    line-height

26. What do we use to change the spacing between letters?

    letter-spacing

27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?

    uppercase, lowercase, capitalize

28. How do I add a 1px border around my div that is dotted and black?

    border: 1px dotted black;

29. How do I select everything on the page? 

    You can use a wildcard selector * or in some cases can use the body tag or other wrapper you may have used. 

30. How do I write a comment in CSS?

    // for a line or /* */ for a section. 

31. How do I find out what file I am in, when I am using the command line? 

    chdir or cd

32. Using the command line - how do I see a list of files/folders in my current folder?

    ls

33. How do I remove a file via the command line? Why do I have to be careful with this? 

    rmdir -r   If you are not in the right folder you could delete system files that will prevent your computer from booting. 

34. Why should I use version control? 

    Version control will help everyone working on a project by allowing to pull a version down from a repository so no one is working on the same idea, and to see what others are working on so you can build onto what they have created.

35. How often should I commit to github?

    You should commit after you have finished a section of code. 

36. What is the command we would use to push our repo up to github? 

    git push origin "sara-doolin"

37. Walk me through Lambda's git flow. 

    fork, add collab, clone
    open git bash
    cd into the directory where you want to store your file
    git clone (link of clone from repository)
    cd into directory that is created
    git checkout -b "your github name"
    code .

    once a section of code has been completed you:
    git status (optional)
    git add . 
    git commit -m "comment"

    once you are through for the day (or with a large section):
    git push origin "your github name"
    pull request
    merge to master
